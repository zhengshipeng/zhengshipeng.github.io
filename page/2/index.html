rabbitMQ基本概念

发表于 2019-05-31 | 分类于 消息队列

名词解析

server(broker)：RabbitMQ Server就是Message Broker。接收和分发消息端的应用，实现AMQP消息队列和路由功能的进程。

Virtual Host：虚拟概念，权限控制组，一个Virtual Host 里面可以有若干个Exchange和Queue，权限控制的最小粒度是Virtual Host。

Exchange：接收生产者发送的消息，根据分发规则，匹配查询表中的routing key，分发消息到queue中。Exchange Type 决定了Exchange路由的消息行为，三种类型：direct，Fanout，Topic。
